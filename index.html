основные команды для клана

Главные команды

Создать клан: /clan create <сокращение> <название> Примечание: сокращение от 2 до 6 символов, а также сокращение и название может содержать только символы латинского алфавита и цифры от 0 до 9. Распустить клан: /clan disband

Пригласить в клан игрока: /clan invite <ник_игрока> Принять приглашение в клан: /clan accept Отклонить приглашение в клан: /clan decline Покинуть клан: /clan resign

Поставить клановый хоум: /clan sethome Телепортироваться на клановый хоум: /clan home

Информация

Список кланов: /clan list Лучшие игроки сервера: /clan leaderboard

Узнать информацию о клане: /clan info <сокращение> Узнать информацию об участниках клана: /clan stats <сокращение> Узнать информацию об игроке: /clan player info <ник_игрока> Просмотреть список участников клана: /clan roster <сокращение>

Координаты участников вашего клана: /clan coords

Дипломатия

Предложить альянс: /clan ally invite <сокращение> Принять предложение об альянсе: /clan ally accept Отклонить предложение об альянсе: clan ally decline Распустить альянс: /clan ally remove <сокращение>

Общение

Перейти в глобальный чат сервера: /clan chat global Перейти в клановый чат: /clan chat clan Перейти в чат альянса: /clan chat ally

Отправить сообщение в глобальный чат сервера: /clan chat global <сообщение> Отправить сообщение в клановый чат: /clan chat clan <сообщение> Отправить сообщение в чат альянса: /clan chat ally <сообщение>

Игнорировать клановый чат: /clan chat ignore clan Игнорировать чат альянса: /clan chat ignore ally

Экономика

Просмотреть баланс вашего клана: /clan bank balance Положить деньги на счёт клана: /clan bank deposit <количество> Снять деньги с кланового счёта: /clan bank withdraw <количество> Расценки на некоторые клановые действия: /clan price

Ранжирование

Список всех имеющихся в клане рангов: /clan rank view Права, которые даёт определённый ранг: /clan rank view <ранг> Список всех прав и их описания: /clan rank permission view

Присвоить игроку ранг: /clan player setrank <ник_игрока> <звание>

Создать новый ранг: /clan rank create <звание> <права> Переименовать ранг: /clan rank rename <звание(текущее)> <звание(присваиваемое)> Удалить ранг: /clan rank remove <звание>

Добавить рангу права: /clan rank permission add <звание> <права> Установить рангу права: /clan rank permission set <звание> <права> Отнять у ранга права: /clan rank permission remove <звание> <права>

Прочеем

Переключить возможность атаковать своих (для клана): /clan friendlyfire clan Переключить возможность атаковать своих (для себя): /clan friendlyfire personal

Изменить цвет кланового тега: /clan tag color <цвет> Примечание: цвет прописывайте по-английски. Например: green, blue, и т.п.
